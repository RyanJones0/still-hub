How to add product images

01 — AI Generation

Generate with Midjourney

Prompt: "flat lay clothing photography, [piece name], [colour], chalk background, natural light, editorial, no text, premium brand"

Use —ar 3:4 for the right ratio. Download at max resolution.

02 — File naming

Drop into /images/products/

Create a folder at still/images/products/ and name files exactly: six-panel-graphite.jpg wool-cap-navy.jpg foundation-tee-chalk.jpg five-pocket-sand.jpg

03 — Done

Images appear automatically

No code changes needed. The page checks for each image at the right path — when the file is there, it shows. When it's not, the colour placeholder shows instead.

Add one colourway at a time as you shoot.
